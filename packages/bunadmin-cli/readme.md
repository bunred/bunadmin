# bunadmin-cli

## Install

```bash
$ npm install --global bunadmin-cli
```


## CLI

```
$ bunadmin --help

Usage
  Create a new project
  $ bunadmin new {name}

Options
    --plugin  with demo plugin
    --doc     with demo document

Examples
  $ bunadmin new my-dashboard
  Your project "my-dashboard" has been created.
```
