## ConfirmDialog

<ConfirmDialog
  doFunc={async () => {
    await notice({
      title: "Confirmed",
      content: "This is a ConfirmDialog test"
    })
  }}
  openModal={true}
  title="Demo Dialog Title"
  msg="This is an example of content, Click `CONFIRM` will active a Notice"
/>
