# Deployment

Enter the path `packages/bunadmin`, and create or modify the file `.env.production` to set [environment variables](/docs/utils/environment-variables).

## Build and export static HTML

```shell
yarn build && yarn export
```

**_This command will generate the `out` directory._**

## Docker build and push

```shell
docker build -t registry:version .
docker push registry:version
```

**_If you need environment variables that are valid at runtime, please use the Dockerfile in the root directory._**

**_`e.g/docker-composer.yml`_**

```yml
version: "3.6"

services:
  bunadmin:
    image: bunadmin:v1.1.0-beta.1.4
    restart: always
    ports:
      - "3000"
```
